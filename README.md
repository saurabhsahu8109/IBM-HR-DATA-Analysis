IBM HR Analytics — Employee Attrition Analysis
📌 Project Overview

This project focuses on analyzing employee attrition using Machine Learning and Data Analytics techniques. The goal is to identify the major factors affecting employee turnover and build predictive models that help organizations improve employee retention strategies.

The project includes:

Data Cleaning & Preprocessing
Exploratory Data Analysis (EDA)
Feature Engineering
Machine Learning Model Building
Attrition Risk Prediction
Workforce Performance Analysis
Data Visualization Dashboard

The analysis was performed using Python libraries such as Pandas, NumPy, Matplotlib, Seaborn, and Scikit-learn.

🎯 Objectives
Analyze employee attrition patterns
Identify high-risk employees
Understand key factors behind employee turnover
Build a predictive model for attrition prediction
Generate actionable HR insights

📊 Key Insights
Employee Attrition Trends
Employees working overtime showed significantly higher attrition.
Employees with low job satisfaction had a higher probability of leaving.
Early-stage employees (0–1 year) had the highest attrition rate.
Low-income employees were more likely to resign.
Workforce Insights
Better work-life balance reduced attrition.
Higher job involvement improved employee retention.
Training sessions positively impacted employee performance.
